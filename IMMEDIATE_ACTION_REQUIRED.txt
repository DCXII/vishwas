                         IF YOU'RE STILL STUCK
================================================================================

1. Check basic connectivity:
   $ ping 8.8.8.8
   $ nslookup google.com

2. View Docker daemon info:
   $ docker info | grep -i dns

3. Check logs:
   Linux:   $ sudo journalctl -u docker -n 50
   macOS:   $ log stream --predicate 'process == "Docker"'

4. Try resetting Docker:
   macOS/Windows: Settings ??? Troubleshoot ??? Reset
   Linux: sudo apt remove docker.io && sudo apt install docker.io
